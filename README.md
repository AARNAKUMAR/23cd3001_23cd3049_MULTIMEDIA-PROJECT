# MediaOdyssey - Interactive Timeline Journey

An immersive educational web application that takes users through the evolution of media from ancient times to the future through an interactive timeline experience.

## ğŸ¯ Features

- **Interactive Timeline**: Navigate through 14 levels spanning the history of media
- **Video Integration**: Watch curated videos for each historical period
- **Quiz System**: Test your knowledge with 5-6 questions per level
- **Fun Facts**: Discover 3 interesting facts for each day/level
- **Progress Tracking**: Monitor your journey completion and quiz performance
- **Badge System**: Earn badges as you complete levels
- **Beautiful UI**: Black background with cyan and golden accents, starfield effects
